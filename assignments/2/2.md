Write a simple GUI in PyQt5, that:

- uses the left mouse button for clicking and adding red points to some image or canvas (training data for the RED class);
- uses the right mouse button for clicking and adding blue points to some image or canvas (training data for the BLUE class).
- has an edit box to specify the number of hidden layers and their sizes and a button to construct such a network: 2 neurons in the input layer (relative coordinates of x and y), the specified hidden layers, and 2 output neurons classifying into the RED class or the BLUE class.
- has an edit box to specify the number of epochs to train and a button that trains the network for an additional number of epochs.
- after the training, each pixel in the image (canvas) except the training data (clicked points) is classified, and its RGB value is determined according to the predicted probabilities for the RED and BLUE classes (i.e., some pixels could be red, blue, violet (high probability for both BLUE and RED classes), black...).

All the team members will then receive the same points according to their solution.
